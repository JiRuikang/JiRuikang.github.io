走进围城

发表于 2017-05-01 | 分类于 生活

再见公主

小女孩总觉得婚礼是这个世界上最重要的事情，所有的女孩都是公主，都该嫁给一位王子，走进童话般的婚礼。
我八岁的小表妹问我：“你选择的是哪个王子”，我想自己也过了“大房花泽类、二房道明寺”的年纪，没什么不好开口的，我回答：“他叫xx”。

我明白，我并不是许先生的公主，起码不是一辈子的公主。小女孩唯独可以是父母的小公主，这样才不会吃了公主病的苦头。
我有公主般修长细腻的双手，母亲总是一边嚷嚷：“长这么大了，一点家务也不会做”，一边絮叨：“这么久才回来一次，学习工作那么累，回来了就好好休息，什么也不要做”。我的父亲干脆连母亲的这点纠结也没有，我回家啃老一直是他莫大的心愿。
然而我怎么忍心，小时候，母亲的双手何尝不是修长细腻，通宵达旦是父亲常有的拼搏，二十五年了，他一定累了。如果有岁月静好，想想是谁在为自己负重前行。
公主长大会变成女王，即便不是女王，即便清纯不再，成熟未满，要成立自己的家庭，第一件事就是对公主的身份说再见。十指不沾阳春水，今来也要做汤赓。嫁人，是放下娇贵矜持的时刻，此刻的掌声属于父母。

何为爱情

拥有同一个青春，就是最好的爱情，我一直这样以为。
人人都想嫁给爱情吧？我们认识的晚，没关系，我们补过青春。

我们唱歌、游泳、吃烧烤，还跑到重庆跨年，在解放碑前，把自己淹没在漫山遍地的大学生里，吹凉凉的风，看热闹的人群，等零点的钟声。
我自然没有真的等到零点，25岁了，睡眠比热闹重要。零点之后窗外越来越吵，跨年的大学生们回来了，我被他们的活力和朝气吸引，趴在窗户边凝望他们。看着他们一群一群的走来，抬着成箱的啤酒，提着大把的烤串。
不知道为什么，从那一刻开始，我就不再想补过青春这件事了。

我跟许先生工作之后相识，但我们的确同一所高中，同一届实验班，同样的高考分数，想到这里，又觉得我们还算有些革命情谊。
没有最好的爱情，只有相伴走过的岁月，人生那么长，世界那么大，错过一些又怎样。

旅行-重庆

发表于 2017-01-02 | 分类于 生活

当代人文

重庆是山城，首先一个山字，就意味着发展受交通限制。打车直线距离显示10分钟就能到，可能要开1个钟头，路都是曲流拐弯高高低低，住惯了平原的我心烦的很，还被迫买一双运动鞋。
如此山城，亦有地铁，它有时是地铁有时是轻轨，有一站直接穿过了一栋楼，有趣的很，在电影《从你的全世界路过》里多次上镜，算是网红地铁吧。
正是因为交通的限制，更显得重庆的繁华弥足珍贵。从城市热力图来看，重庆算是内陆明亮的一颗星了。渝中区CBD高楼林立，商铺营业到很晚，跨年夜漫天遍地大学生，解放碑广场犹如白昼。重庆不辱没直辖市的称号，房价却不高，据当地人说得益于薄执政时大力推广公租房的政策。
元旦的重庆雾蒙蒙的，一腔心事延绵的样子，不知是嘉陵和长江的无语凝噎，是歌乐山革命英烈的哀怨，还是文艺青年们不可释怀的的忧伤，总是一副清淡寡欢的样子。

美食

火锅

北方的叫涮羊肉，重庆的才叫火锅，火锅必点菜品一定不同于涮羊肉，我来推荐几个：竹笋、竹荪、毛肚、鸭血、鸭肠、菌菇。街上常碰见挑着扁担卖竹笋的货郎，我没有考证元旦是否正是竹笋收获的季节，总之我吃到的是这辈子最好吃的竹笋，嘎嘣脆，爽口极了。
重庆火锅必配油碟，油里还要放勺蒜，从锅里夹一筷子美食，往油碟里一蘸，即降温又提味。
另外，一定要点一盘当地的酥肉，不同于井格这类连锁店，当地酥肉里搅和了许多麻椒，别具一番风味。
我去吃火锅的地方比较随意，是从美团上查的评分较高，离住所又近的地方，据当地人说朝天门的火锅比较受游客欢迎。

小吃

当地出租车司机推荐了小滨楼，在一个商场的n层，环境优雅，顾客很多。重庆小面、酸辣粉、红油抄手等等四川小吃在这里都可以吃到，跟成都的龙抄手大概是一个意思。

串串

路边小推车的串串，写到这里不由的吞了个口水，又像烧烤又像铁板烧，很辣很过瘾，不知是不是因为用的佐料更地道，小摊会营业到夜里一两点，完全不辱没重庆不夜城的名号。
